Projet Arduino IoT : ESP32 et Arduino Uno

Ce projet combine une carte Arduino Uno et une ESP32 pour créer un système IoT complet. Le but est de collecter des données depuis un capteur de température, les envoyer à Arduino IoT Cloud, et pouvoir piloter des objets connectés à distance.

La programmation est réalisée en C++, et Arduino IoT Cloud est configuré pour créer une application web/mobile permettant de visualiser les données en temps réel et de contrôler des dispositifs comme des lampes ou relais.

Usage proposé

Hypothèse

Le projet permet de surveiller la température via un capteur connecté à Arduino Uno ou ESP32 et d’envoyer ces données vers Arduino IoT Cloud. L’utilisateur peut recevoir les mesures sur une application et envoyer des ordres pour contrôler des objets connectés.

Applications concrètes

Exemples d’utilisation :

  • Surveillance de la température dans une maison ou un laboratoire
  • Contrôle à distance d’une lampe, ventilateur ou chauffage
  • Création d’une application IoT pour visualiser les données et contrôler les objets connectés
  • Projets éducatifs pour apprendre l’IoT et la programmation C++

Ces applications combinent automatisation, confort, sécurité et apprentissage pratique de l’IoT.

Description des composants

Arduino Uno

  • Microcontrôleur ATmega328P
  • Compatible avec capteurs analogiques et numériques
  • Programmation en C++ via Arduino IDE
  • Connexion série pour communiquer avec ESP32 ou ordinateur

ESP32 DevKit V1

  • Microcontrôleur avec WiFi et Bluetooth intégrés
  • Compatible avec Arduino IoT Cloud
  • Lecture de capteurs et contrôle d’actionneurs

Capteur de température

  • Mesure la température ambiante
  • Connecté à Arduino Uno ou ESP32
  • Envoi de données vers le cloud pour visualisation et analyse

Arduino IoT Cloud

  • Plateforme pour recevoir et visualiser les données des objets connectés
  • Création d’une application web/mobile pour pilotage et monitoring
  • Permet d’envoyer des ordres de contrôle aux cartes connectées

Schématisation matérielle

Expérimentations

Expérimentation 1 : Connexion au WiFi

Objectif : Configurer ESP32 pour se connecter à Arduino IoT Cloud.

Résultat : La carte se connecte correctement et les données peuvent être envoyées au cloud.

Expérimentation 2 : Lecture de température

Objectif : Lire les mesures du capteur de température via Arduino Uno ou ESP32 et les envoyer vers Arduino IoT Cloud.

Résultat : Les valeurs sont affichées en temps réel sur l’application IoT Cloud.

Expérimentation 3 : Contrôle d’objets connectés

Objectif : Envoyer des ordres de contrôle depuis Arduino IoT Cloud pour allumer/éteindre des lampes ou actionneurs.

Résultat : Les objets réagissent correctement aux commandes envoyées via l’application.
